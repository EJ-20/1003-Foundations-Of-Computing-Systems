Exercise 1: Rectangle

Complete these methods for the Rectangle class, which can be found in Code for Practice Exercises.

  • perimeter(self)

    • returns the perimeter of the rectangle

  • overlaps(self, other)

    • returns True if self overlaps the other rectangle, otherwise False

    • treat coincident lines as overlapping, so a.overlaps(a) returns True

    • treat rectangles as overlapping if one contains the other

  • contains(self, other)

    • returns True is self contains the other rectangle within its boundaries, otherwise False

    • ensure that a.contains(a) returns True

# Exercise 1: Rectangle.  Methods to add:

    # Return the perimeter of self.
    def perimeter(self):
        return 2 * (self.w + self.h)

    # Return True if self overlaps other, otherwise False.
    def overlaps(self, other):
        if (self.x <= other.x + other.w and
        self.x + self.w >= other.x and
        self.y <= other.y + other.h and
        self.y + self.h >= other.y): return True
        return False
        # (Why does this work? Go and find out!)

    # Return True if other is completely inside of self, otherwise False.
    def contains(self, other):
        if (self.x <= other.x and self.x + self.w >= other.x + other.w and self.y <= other.y and self.y + self.h >= other.y + other.h): return True
        return False

Exercise 5: Fractions

Extend your code from Exercise 4 by adding the special (or 'magic')  method __eq__(self, b).  This returns True if a == b, and False otherwise.

To avoid rounding errors, this method should not compare two fractions by converting them to decimal (Why?  Go find out!).  Instead it should decide if two fractions are equal by comparing the numerators and denominators of the simplest forms of each fraction.

# Exercises 3 to 5: Fractions. Rational class and test client:

class Rational:
    
    def __init__(self, x, y):
        self.numerator = x
        self.denominator = y
    
    def __add__(self, b):
        n = self.numerator * b.denominator + b.numerator * self.denominator
        d = self.denominator * b.denominator
        return Rational(n, d)
    
    def __sub__(self, b):
        n = self.numerator * b.denominator - b.numerator * self.denominator
        d = self.denominator * b.denominator
        return Rational(n, d)
    
    def __mul__(self, b):
        n = self.numerator * b.numerator
        d = self.denominator * b.denominator
        return Rational(n, d)
    
    def __abs__(self):
        n = abs(self.numerator)
        d = abs(self.denominator)
        return Rational(n, d)
    
    def __str__(self):
        return str(self.numerator) + "/" + str(self.denominator)
    
    def simplest(self):
        # find greatest common divisor
        # (note: there is a far more elegant recursive method for this - look it up!)
        gcd = 1
        max = self.numerator if abs(self.numerator) > abs(self.denominator) else self.denominator
        for index in range(1, max):
            if self.numerator % index == 0 and self.denominator % index == 0:
                gcd = index
        
        # return the reduced fraction
        n = self.numerator//gcd
        d = self.denominator//gcd
        if d < 0:
            n = 0 - n
            d = 0 - d
        return Rational(n, d)

    def __eq__(self, b):
        f1 = self.simplest()
        f2 = b.simplest()
        return f1.numerator == f2.numerator and f1.denominator == f2.denominator

# test client
def main():
    third = Rational(1,3)
    twoFourths = Rational(2,4)
    half = Rational(5,10)
    negDenominator = Rational(3,-7)
    added = third + twoFourths
    subtracted = third - half
